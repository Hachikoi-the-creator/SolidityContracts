## How to install hardhat now
- mkdir hardhat-project
- cd hardhat-project

- npm init -y
- npm i -D hardhat
- npx hardhat
- npm i -D @nomiclabs/hardhat-waffle ethereum-waffle chai @nomiclabs/hardhat-ethers ethers
